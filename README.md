# Web3 Bridge Simulator Basic

This project helps beginners learn how blockchain bridges work.

You will learn:
- What a blockchain bridge is
- How to connect a wallet
- How to lock tokens on one chain
- How to unlock tokens on another chain
- How cross-chain apps work

## Requirements
- Node.js installed
- MetaMask browser extension
- Basic JavaScript knowledge

## Setup

1. Clone the repo
git clone https://github.com/yourusername/web3-bridge-simulator-basic.git

2. Install packages
npm install

3. Run the app
npm start

4. Open browser
http://localhost:3000

## Project Structure

- index.html = Main webpage  
- app.js = Frontend logic  
- bridge.js = Bridge contract functions  
- web3.js = Blockchain connection  
- deploy.js = Contract deploy script  
- config.js = Network settings  
- server.js = Local server  
- package.json = Project info  
- utils.js = Helper functions  
- .env.example = Environment example  

## Features
- Connect wallet  
- Lock tokens on Chain A  
- Unlock tokens on Chain B  
- Show transaction hash  
- Simple cross-chain flow  

## Learning Goal
This project explains how blockchain bridges move value between networks.
